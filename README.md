# 栄ガーデン シフト管理（スタンドアロン版）

ブラウザだけで動く単一ファイルのシフト管理アプリ。git で配布し、PC / スマホ / どの端末からでも使えます。

## 使い方
- ローカル: `index.html` をブラウザで開く
- 簡易サーバ: `npx serve .` または `python3 -m http.server 8080`
- 公開: GitHub Pages で Settings → Pages → main / root を選択

## データ
ブラウザの localStorage に保存。右上の ↓ で JSON 書き出し、↑ で読み込み。端末間同期は JSON で手動同期。

## 主な機能
月間カレンダー（基本+例外）／日付タップで休み・追加出勤を追加／個人別 月間集計／従業員マスタの編集／A4横の印刷ビュー
